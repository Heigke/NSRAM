Q: What did you work on in the last two years that makes you proud?

I created nsram (pip install nsram) — the first open-source simulator for the NS-RAM neuromorphic device published in Nature (Pazos et al., 2025). What started as curiosity about a paper became an active collaboration with the original authors at NUS. The library now has 16 modules covering device physics through reservoir computing, and I demonstrated Forward-Forward learning at 14-level weight quantization with zero accuracy loss — something nobody had published before. I also investigated continual learning by mapping Elastic Weight Consolidation to actual device physics (oxide charge trapping), reducing catastrophic forgetting from 69 percentage points to 22. None of this was my job — it was pure research drive, built evenings and weekends with Claude Code as a co-engineering partner.
